Membership features to sell access to content on your WordPress e-commerce site

== Description ==

With Cart66 Cloud Members you can sell memberships and subscriptions to give your customers access to pages, posts, or even entire post categories on your WordPress website. A person can have an unlimted number of memberships. You can even drip access to your content based on how long a person has had their membership.

== Features ==

Here are some examples of what you can sell using Cart66 Cloud Members:

* membership access to categories of your blog posts
* online courses
* access to wholesale products
* premium support content
* sections of a page can be protected like embedded videos

== Installation ==

= Minimum Requirements =

* WordPress 3.2 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

= Automatic installation =

You can install Cart66 Cloud Members directly from the WordPress.org plugin directory without having to worry with manually transferring files to your server. Automatic installation is the easiest option for installation. Let WordPress install Cart66 Cloud Members for you.

To install Cart66 Cloud Members, log in to your WordPress admin panel, go to the Plugins menu and click the "Add New" link.

Type "Cart66 Cloud Members" in the search field then click the "Search Plugins" button. You will see the Cart66 Cloud Members plugin in the search results. To install, simply click the "Install Now" link. After confirming that you do want to install the plugin, WordPress will automatically download and install Cart66 Cloud Members directly on your WordPress website.

Once the Cart66 Cloud Members plugin is installed, you may want to visit [our tutorials](http://help.cart66.com) for more information on getting started.

= Manual installation =

If you would prefer to manually install Cart66 Cloud Members, you will need to download the Cart66 Cloud Members plugin and upload it to your web server via FTP. Here are the steps to follow:

1. Download and unzip the plugin file on your computer
2. Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to the plugin directory on your WordPress site (wp-content/plugins/<cart66-cloud-members>).
3. Activate the Cart66 Cloud Members plugin from the "Plugins" menu in your WordPress admin panel.

= Upgrading =

When new versions of Cart66 Cloud Members are released, WordPress will notify you about the available update. You can update to the latest version of the Cart66 plugin by selecting to do an automatic updates. It is always a good idea to backup your site before doing any updates of any kind including updating plugins.

Cart66 Cloud Members will remember all of your settings and products after updating so you will not have to re-enter your membership information or anything else.

== Frequently Asked Questions ==

= Do I need Cart66 Cloud to use the Cart66 Cloud Members Plugin? =

Yes, Cart66 Cloud provides the foundation of selling products securely on your website. Cart66 Cloud Members add the ability to sell memberships through Cart66 Cloud.
